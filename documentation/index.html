Getting Started

Installation
  • Node.js and NPM : You can download Node.js from https://nodejs.org. NPM comes bundled with Node.js.
Install Npm Packages
  • First unzip cobra-template.zip you will find two folders including cobra (this is your root folder). Open this folder with VS Code (or any other you like). And follow step below.
  • NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.
Development server
  • Run npm run dev for a dev server. Navigate to http://localhost:3000 . The app will automatically reload if you change any of the source files.
Generate static project
  • Run npm run generate to generate static project. Generated project will be stored in the dist/ directory.
Build for production and launch server
  • Run npm run build and then npm run start

Customization

How to change The Image & Text In HeroArea Component ?

To change the the image in HeroArea Component replace the image in (static/assets/img/welcome-area) folder with your new image (recommended size 500px * 716px)and replace the yellow highlighted text with (/assets/img/welcome-area/imgName.jpg).
To change the text replace the blue highlighted text with your own



How to change the template icon ?

To change icons in Visit Font Awesome and Simple Line Icons to get more icons.

How to change The Image & In AboutArea Component ?

Open components/AboutArea.vue

to change the about image put your image in (static/assets/img/about-area) folder (recommended size 824px * 1074px) and replace the yellow highlighted text with (/assets/img/about-area/imgName.jpg).

How to change WorksArea Component Images ?

Open components/WorksArea.vue

to change the images put your image in (static/assets/img/works) folder and replace the yellow highlighted text with (assets/img/works/imageName.jpg).
Note: You can edit all images by the same steps.
